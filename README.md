## E-commerce Customer Behavior Analysis
- Objective
Analyze customer behavior to understand user journey and improve conversion.
## Dataset
- ~38K users
- ~47K sessions
- Events: view, cart, purchase

## Tools
- SQL (MySQL)
- Power BI

## Key Metrics
- Conversion Rate: ~2%
- Cart Rate: ~1%
- Revenue: ~$1.1M

## Key Insights
- High drop-off at cart stage → UX or pricing issue
- Electronics is the most viewed category
- Samsung & Apple are top brands

## Recommendations
- Improve checkout UX
- Optimize pricing strategy
- Focus marketing on high-performing categories
